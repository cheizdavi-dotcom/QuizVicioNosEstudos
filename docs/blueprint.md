# **App Name**: StudyFlow Quiz

## Core Features:

- SPIN Selling Quiz: A 5-question quiz designed using the SPIN selling methodology to identify user pain points related to studying.
- Dynamic Question Flow: Automatic progression to the next question upon selecting an answer to ensure a seamless user experience.
- Personalized Result Generation: Generates a custom diagnosis based on the user's response pattern, providing actionable insights.
- Persuasive CTA Integration: Incorporates a compelling Call To Action (CTA) at the end of the quiz to drive conversions to the 'Como Ficar Viciado em Estudar' method.

## Style Guidelines:

- Background: Dark, to provide a modern, engaging feel.
- Primary: Neon Green (#39FF14) for highlights, progress bar, and CTAs to draw attention.
- Accent: A lighter shade of green (#B2FF9E) to complement the primary color. This will be used for subtle UI elements and hover states.
- Body and headline font: 'Inter' for a clean and modern look.
- Centered layout within a modern card design to focus user attention and provide a clean aesthetic.
- Smooth, animated progress bar to visually indicate quiz progression and enhance user engagement.
